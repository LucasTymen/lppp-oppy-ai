Posez votre question sur les démarches, horaires, déchets, état civil ou les services de la ville. L’assistant répond 24h/24 en s’appuyant uniquement sur les informations officielles du site de la mairie.

Démo Concierge IA

Pour afficher le chat : configurez le Chatflow RAG dans Flowise, puis remplacez dans ce fichier (ou via une variable) l’URL du chat par FLOWISE_CHAT_URL.

Exemple : https://votre-flowise.com/api/v1/prediction/xxx ou l’URL d’embed fournie par Flowise.
