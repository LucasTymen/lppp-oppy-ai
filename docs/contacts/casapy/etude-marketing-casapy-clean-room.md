## 1) Marché & concurrence

### 1.2 Concurrence pertinente — **à établir**

## 3) Funnel AARRR (Retail)

### A) Acquisition — **audit**

- **SEO** : pages catégories indexables, filtres crawlables, contenus, maillage, données structurées
- **SEA/Shopping** : constat actuel — **aucune trace Meta/Google** ; flux Merchant Center à vérifier
- **Social** : présence organique, UGC, pins

**KPIs** : sessions, part SEO, part Shopping, CAC/ROAS, CTR SERP, CPM/CPA.

### B) Activation — **audit UX**

Checklist fiche produit : photos HD, vidéos, dimensions, variantes, avis, Q&A ; réassurance livraison/retours/paiement ; prix + frais + délais avant panier.

**KPIs** : PDP→ATC, ATC→checkout, checkout→purchase, abandon panier, scroll depth.

### C) Rétention — **plan**

Post-achat (email, reco, inspiration). **KPIs** : repeat rate 60/90j, CLV, taux d’avis, NPS, retours.

## 4) SWOT — strictement défendable

- **Forces** : uniquement si preuve (page livraison, mentions, avis, labels)
- **Faiblesses** : si observables (absence avis PDP, infos livraison tardives, vitesse mobile…)
- **Opportunités** : génériques (UGC, SEO longue traîne, feed Shopping, bundles)
- **Menaces** : génériques (inflation CPC, marketplaces, seconde main, ruptures)

**Règle** : chaque Force a une **preuve**.
