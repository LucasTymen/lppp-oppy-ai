7 formats visuels (score global, timeline utilisateur, compteur perte, bullet chart, heatmap, waterfall, recovery). Données : TTFB ~3,7 s, LCP 32 s, 54 k€/mois.

1. Score global (tachymètre type Lighthouse)

Une seule note 0–100 : le client voit tout de suite « dans le rouge » sans lire les secondes.

2. Ce que vit l'utilisateur (timeline 0 → 35 s)

La frise montre l'attente réelle : le vide entre les marqueurs = temps perdu. Impact émotionnel fort.

3. Compteur « perte en direct » (Lost Revenue Meter)

Équivalent taxi : chaque seconde qui passe = manque à gagner. Urgence business sans lire un rapport.
