# Dossiers contacts (prospects)

**Règle** : **Un contact = un dossier.** Toutes les données liées à un contact (prospect) sont rangées dans son dossier pour éviter de mélanger les informations.

- **Convention** : un sous-dossier par contact, nommé par un **slug** unique (ex. `societe-dupont_jean`, `cabinet-martin_relance-salon-2025`).
- **Contenu type** : `content_json`, notes, rapport SEO synthèse, pièces annexes (voir `docs/base-de-connaissances/organisation-donnees-contacts.md`).

**Registre** : `docs/contacts/REGISTRE-CONTACTS.md` — liste des contacts ayant un dossier.

**Doc de référence** : `docs/base-de-connaissances/organisation-donnees-contacts.md`
