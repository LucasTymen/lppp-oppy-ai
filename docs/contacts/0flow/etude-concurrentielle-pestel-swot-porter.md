## 2. PESTEL — Le vent dans le dos

- **Politique / Légal** : Le RGPD est leur meilleur ami. Les boîtes ont peur de l'IA « sauvage » ; 0Flow propose un cadre sécurisé.
- **Économique** : En période de crise, les boîtes ne recrutent plus, elles automatisent. C'est le moment idéal.
- **Social** : Peur du remplacement par l'IA. Angle : « Je ne remplace pas l'humain, je supprime les tâches répétitives qui le rendent malheureux. »
- **Technologique** : L'émergence des agents (Auto-GPT, BabyAGI). C'est là que le Python et le Glue Code interviennent.

---

## 3. Les 5 forces de Porter — Leur champ de bataille

- **Concurrence sectorielle (forte)** : Make.com (puissant mais complexe), Zapier (cher), UIPath (très grosses boîtes).
- **Nouveaux entrants (moyenne)** : Des milliers de micro-SaaS d'IA qui sortent chaque jour.
- **Pouvoir des fournisseurs (élevé)** : OpenAI et Google Anthropic dictent les prix des API.
- **Pouvoir des clients (moyen)** : Ils veulent du No-Code mais ont besoin de Code dès que c'est complexe. **C'est là que tu interviens.**
- **Menace des substituts** : Les équipes internes qui développent leurs propres scripts Python (ce que tu sais faire).
