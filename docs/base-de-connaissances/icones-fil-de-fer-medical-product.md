# Icônes fil de fer / ultra thin — médical & produit (gratuit)

**Contexte** : portfolio Yuwell et projets LPPP ; souhaite décliner les icônes en style **fil de fer** (outline, trait fin / ultra thin), secteur médical / produit.  
**Objectif** : recenser ce qui existe de **gratuit** et exploitable (SVG, licence claire).

---

## 1. Health Icons

- **URL** : [healthicons.org](https://healthicons.org/)
- **Style** : Outline (stroke 2 px) et Filled.
- **Licence** : CC0 — utilisation commerciale et personnelle OK ; pas d’attribution requise ; modification et republication OK.
- **Contenu** : 1 700+ icônes santé : blood, body, conditions, devices, diagnostics, medications, nutrition, people, places, PPE, symbols, vehicles, etc. (stéthoscope, seringue, fauteuil roulant, hôpital, etc.).
- **Formats** : SVG, PNG (24 px, 48 px), possiblement JPG ; site, plugin Figma, packages React.
- **Atout** : 100 % orienté santé / médical ; très permissif.

---

## 2. Lineicons (Health & Medical)

- **URL** : [lineicons.com/icons/category/health-medical](https://lineicons.com/icons/category/health-medical)
- **Style** : Line / outline (plus variantes solid, duotone, etc.).
- **Licence** : MIT — gratuit commercial et personnel.
- **Contenu** : 575 icônes Health & Medical (équipements, ambulances, conditions, etc.) ; 2 298+ icônes en gratuit « essential ».
- **Formats** : SVG, PNG ; CDN, NPM, Figma.
- **Atout** : volume médical important, style « line » adapté fil de fer.

---

## 3. Unicons (IconScout) — Medical

- **URL** : [iconscout.com/unicons/free-line-icons/medical](https://iconscout.com/unicons/free-line-icons/medical)
- **Style** : Line et **Thin line** (ultra thin).
- **Licence** : à vérifier sur le site (gratuit avec conditions possibles).
- **Contenu** : ensemble médical en line + thin line.
- **Formats** : SVG, font, React.
- **Atout** : variante **thin line** explicite, proche « fil de fer ».

---

## 4. Lucide Icons

- **URL** : [lucide.dev](https://lucide.dev/)
- **Style** : Outline fin (héritier Feather Icons), minimaliste.
- **Licence** : ISC.
- **Contenu** : 1 000+ icônes ; catégorie médicale (ex. briefcase-medical, heart, etc.), moins exhaustif que Health Icons pour le médical pur.
- **Formats** : SVG, React, Vue, Svelte, etc.
- **Atout** : trait fin cohérent, maintenu, bon pour UI produit / interface.

---

## 5. Icons8 (Health / Healthcare — style Wired / Outline)

- **URL** : ex. [icons8.com/icons/set/healthcare--style-wired](https://icons8.com/icons/set/healthcare--style-wired), [icons8.com/icons/set/health--style-wired](https://icons8.com/icons/set/health--style-wired)
- **Style** : Cute Outline / Wired (ligne).
- **Licence** : gratuit souvent avec attribution ou lien ; usage commercial sans watermark possible sous conditions (vérifier sur le site).
- **Contenu** : santé, soins, équipements, hôpital, etc.
- **Formats** : PNG, SVG.
- **Atout** : style « wired » proche fil de fer ; à valider licence pour usage commercial sans attribution.

## Synthèse rapide

| Besoin | Recommandation |
|--------|----------------|
| **Maximum médical, 0 attribution** | Health Icons (CC0, outline 2 px). |
| **Très fin (ultra thin) + médical** | Unicons Thin line (vérifier licence). |
| **Médical + volume + line** | Lineicons (MIT, 575 health & medical). |
| **UI produit, trait fin général** | Lucide (ISC, style Feather). |
| **Style wired / outline, variété** | Icons8 (vérifier licence). |

Pour le **portfolio Yuwell** : privilégier **Health Icons** (outline) et **Lineicons** (line) pour le médical ; **Lucide** ou **Unicons thin line** si on veut un rendu encore plus « fil de fer » sur des symboles plus génériques. Toujours appliquer la **charte graphique** (couleurs corporate / gamme, contraste) aux icônes intégrées.
