# Rapport synthétique Casapy — Version présentation

**Date** : 2026-01-30  
**Responsable** : **Rédacteur** (Accountable)  
**Statut** : 🟡 En cours

**Brief** : `docs/contacts/casapy/brief-rapport-synthetique-presentation.md`

---

## Objectif

Préparer un **rapport synthétique** pour présentation au **responsable placement** (école), au **patron / responsable alternance** (ou tout référent) rencontré. Le rapport doit **parler**, être **suffisamment documenté** sans entrer dans tous les détails, et s’appuyer sur des **illustrations très parlantes et à propos**.

---

## Périmètre

- **Public** : responsable placement, patron / responsable alternance, référents (entretien, suivi alternance).
- **Contenu** : contexte Casapy, diagnostic synthétique, chiffres clés (2–3), plan d’action (3–5 priorités), illustrations (infographies, schémas, tableaux).
- **Source** : rapport complet interne Casapy (`rapport-seo-complet-casapy-interne.md`) et `SEO_Casapy_rapport_performances_et_diagnostique.md` + données CSV si besoin.

---

## Répartition des rôles

### Rédacteur (Responsable — Accountable)
- [ ] Structurer le rapport synthétique (sections : contexte, diagnostic, chiffres clés, plan d’action, annexes / visuels).
- [ ] Rédiger le contenu : clair, percutant, compréhensible par un non-technique.
- [ ] Coordonner les contributions (référent technique, Infographiste, chargé chiffres).
- [ ] Livrer : `docs/contacts/casapy/rapport-synthetique-casapy-presentation.md` (et export PDF si besoin).

### Expert SEO (Référent technique — Consulted)
- [ ] Valider la justesse technique des formulations (TTFB, indexation, canoniques, hébergement, plan d’action).
- [ ] Éviter sur-simplifications trompeuses ou erreurs factuelles.
- [ ] Relire les parties techniques du brouillon.

### Infographiste (Responsible — illustrations)
- [ ] Produire des **illustrations très parlantes et à propos** (infographies, schémas, KPI visuels).
- [ ] S’appuyer sur le brief InfographicCraft (THEME : rapport synthétique Casapy ; AUDIENCE : placement / patron / référent ; DATA : chiffres du rapport interne / diagnostique).
- [ ] Déposer les visuels dans `docs/contacts/casapy/` (ex. `infographie-rapport-synthetique-casapy.html`).
- [ ] Pas d’invention de chiffres ; données fournies par le Rédacteur ou le chargé chiffres.

### Chargé chiffres / mathématiques (Data Analyst ou Growth Analyst — Responsible)
- [ ] Fournir **tableaux et schémas chiffrés** à l’appui du rapport (manque à gagner, scénarios conversion/panier, hypothèses).
- [ ] Sourcer les données (rapport interne, CSV, `SEO_Casapy_rapport_performances_et_diagnostique.md`) ; pas d’invention.
- [ ] Formater pour intégration dans le rapport ou pour l’Infographiste.

### Chef de Projet (Consulted)
- [ ] Valider le brief et la cible (placement, patron, référent).
- [ ] Valider le livrable final avant présentation.
