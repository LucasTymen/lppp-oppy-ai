# Ressources utilisateur LPPP

Espace dédié aux **données et informations** que tu transmets au Conseiller : textes, images, ébauches, textes d’études ou toute autre ressource utile au projet.

## Pourquoi c’est ici

- **Un seul endroit** : tout ce que tu envoies au Conseiller est enregistré ici, de façon structurée.
- **Disponible partout** : le registre (`REGISTRE-RESSOURCES.md`) et les dossiers sont référencés dans le registre des agents ; tous les agents peuvent s’y référer quand c’est pertinent.
- **Retrouvable facilement** : nommage clair des fichiers + tableau dans `REGISTRE-RESSOURCES.md` (date, type, titre, fichier, usage).

## Structure

| Dossier   | Contenu |
|-----------|---------|
| `textes/` | Contenus, notes, documents rédigés |
| `images/` | Visuels, captures, schémas, références |
| `ebauches/` | Brouillons, wireframes texte, maquettes préliminaires |
| `etudes/` | Textes d’études, recherches ayant servi à élaborer un système ou un outil |
| `autres/` | Autres formats (PDF, etc.) ou ressources diverses |

## Comment ajouter une ressource

Tu transmets au **Conseiller** (point d’entrée) : texte, fichier image, ébauche, document d’étude, etc. Le Conseiller :

1. Enregistre le fichier dans le bon dossier avec un nom explicite.
2. Ajoute une ligne dans `REGISTRE-RESSOURCES.md` (date, type, titre, chemin, usage/contexte).

Aucune action requise de ta part côté dépôt : tu donnes l’info, le Conseiller la range.

## Référence

- **Registre des ressources** : `docs/ressources-utilisateur/REGISTRE-RESSOURCES.md`
- **Règle Conseiller** : `.cursor/rules/conseiller.mdc`
